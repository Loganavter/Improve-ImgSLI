## Списки и всплывающие панели

Выпадающие списки открывают панели поверх главного окна. Работаете внутри, затем закрываете кликом снаружи или `Esc`. Жесты — в [Кнопки и контролы](help://ui.buttons); стиль подписей имён — в [Сравнение](help://comparison).

### Менеджер списков {#list-manager}

В сессии {{tr:workspace.session_types.image_compare}} у каждой стороны есть выпадающий список. Клик открывает панель менеджера этой стороны (строки, рейтинги, перетаскивание). Если список пуст, панель не открывается. Повторный клик по тому же списку, клик снаружи или выбор строки закрывают её. Пока открыто переименование или свойства, панель не закрывается из‑за потери фокуса.

### Скролл {#scroll-lists}

- **Колесо на списке** — шагает текущее изображение на этой стороне, не открывая панель.
- **Колесо на холсте** — шагает сторону под курсором; `Shift` + колесо — оба списка. Полные правила: [Сравнение → Скролл по картинкам](help://comparison#scroll-images).
- **Колесо на рейтинге** — меняет только оценку строки, не текущий индекс.

### Строки {#rows}

- **Выбор** — клик по строке делает её текущей; панель закрывается, холст обновляется.
- **Порядок** — перетаскивание внутри одного списка.
- **Перенос** — вынесите строку за пределы одиночной панели (двойной режим), затем бросьте на другой список.
- **Путь** — наведите на обрезанное имя, чтобы увидеть полный путь.

### Рейтинг {#rating}

У каждой строки есть чип рейтинга. Минус / плюс или колесо по чипу меняют оценку, не закрывая панель.

### Контекстное меню {#context-menu}

- **Строка списка** — переименовать, копировать путь, свойства или удалить.
- **Холст (текущий кадр)** — то же меню плюс дублировать; переименование — только из списка.

Свойства открывают [Свойства изображения](help://image_properties).

### Кнопки тулбара {#quick-list-actions}

- **Добавить файлы** — кнопка рядом с каждым списком; добавляет только на эту сторону.
- **Поменять** (`X`) — короткий клик меняет текущую пару; удержание — оба списка целиком.
- **Удалить** — короткий клик убирает текущий кадр на своей стороне; удержание очищает весь список этой стороны.

### Загрузка {#loading}

Перетаскивание файлов в окно спрашивает, в какой список положить. `Ctrl+V` вставляет изображение из буфера и может показать оверлей стороны — см. [Файлы и проекты](help://file_management).

### Панель настроек подписей {#toolbar-flyouts}

{{tr:image_compare.action.text_settings}} (или ПКМ по {{tr:image_compare.action.file_names}}) открывает панель: размер, начертание, цвета, размещение. Закрытие — `Esc` или клик снаружи. Цвет разделителя — через палитру, не через панель: [Сравнение](help://comparison). Панели лупы: [Лупа](help://magnifier).

### {{tr:workspace.session_types.multi_compare}} {#multi-compare}

В сессии {{tr:workspace.session_types.multi_compare}} нет двойного менеджера списков — изображения кладут в ячейки сетки. Настройки текста подписей по‑прежнему открывают панель (без радио размещения, как в {{tr:workspace.session_types.image_compare}}). Подробнее: [{{tr:workspace.session_types.multi_compare}}](help://multi_compare).
