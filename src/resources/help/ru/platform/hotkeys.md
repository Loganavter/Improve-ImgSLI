## Горячие клавиши

Ниже — аккорды по умолчанию. Большинство переназначаются в {{tr:menu.settings}} → {{tr:settings.keyboard}}. `WASD` и `Space` на холсте фиксированы и не ремапятся.

### Сначала поиск {#discover}

`Ctrl+Shift+P` и имя действия — быстрее списка. Запустите из палитры или откройте {{tr:action.palette.learn_more}}. `F1` может подсветить действие, если фокус помечен.

### Платформа {#platform}

- `Ctrl+,` — Настройки
- `Ctrl+F1` — Справка
- `Ctrl+N` — Выбор сессии / новая сессия
- `Ctrl+Shift+P` — {{tr:menu.find_action}}
- `Ctrl+V` — Вставка изображения
- `Ctrl+Shift+O` / `Shift+S` / `Ctrl+Shift+S` — Открыть / сохранить / сохранить проект как (`.imgsli`)
- `Ctrl+Q` — Выход

Подписи зависят от языка; ремап — в группе platform на странице клавиатуры.

### {{tr:workspace.session_types.image_compare}} {#image-compare}

- `M` / `F` — Лупа / заморозка
- `N` / `D` — Подписи / видимость разделителя
- `H` / `C` — Режим разницы / канал
- `X` — Поменять
- `R` / `Ctrl+E` — Запись / видеоредактор
- `Ctrl+S` — Быстрое сохранение
- `WASD` / `QE` / `Space` — Движение лупы / расстояние / превью стороны (фиксировано)

Подробнее: [Сравнение](help://comparison), [Лупа](help://magnifier), [Видеоредактор](help://video).

### {{tr:workspace.session_types.multi_compare}} {#multi-compare}

- `Ctrl+O` — Добавить изображения (если назначено)
- `D` — Видимость сетки
- `Ctrl+S` — Быстрое сохранение
- `Esc` — Выйти из фокуса слота

См. [{{tr:workspace.session_types.multi_compare}}](help://multi_compare).

### Окно видеоредактора {#video-editor}

Пока редактор открыт: `Space` — воспроизведение / пауза, `Ctrl+Z` / `Ctrl+Y` — отмена / повтор, `Delete` / `Backspace` — удалить выделение. Полный сценарий: [Видеоредактор](help://video).
