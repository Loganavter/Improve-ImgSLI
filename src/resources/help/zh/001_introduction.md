## 快速开始

本页用于快速上手：加载一对图像，进行比较，需要时启用放大镜，然后保存结果。

### 基本流程 {#basic-workflow}
- 将图像添加到左侧和右侧列表。
- 通过下拉菜单或鼠标滚轮切换当前图像对。
- 使用分割线或放大镜比较图像。
- 需要时启用指标、文件名标签或差异模式。
- 使用 **保存结果** 或 `Ctrl+S` 保存结果。

### 接下来该看哪里 {#where-to-go-next}
- **[文件管理](help://file_management)**：加载、拖放、剪贴板粘贴、列表操作。
- **[画布导航](help://view_navigation)**：缩放、平移、快速预览。
- **[比较](help://comparison)**：分割线、指标、文件名标签、无放大镜流程。
- **[放大镜](help://magnifier)**：捕获区域、合并模式、多实例。
- **[导出与视频](help://export)**：保存图像、录制会话、视频编辑器。
- **[设置](help://settings)**：性能、质量、语言和系统行为。

### 什么时候打开设置 {#when-to-open-settings}
- 当预览性能太重时。
- 当你需要更改语言、主题或界面字体时。
- 当你想在 **初学者 / 高级 / 专家** 界面模式之间切换时。
- 当你想启用自动 PSNR / SSIM 计算时。
- 当你需要调整放大镜行为或响应速度时。

### 系统托盘与通知 {#system-tray-and-notifications}
- 如果系统支持托盘，应用可以在那里提供快捷操作。
- 托盘菜单可以显示或隐藏窗口、打开最后保存的文件或文件夹，以及退出应用。
- 保存成功后可能会出现系统通知。
- 通知可在 **设置** 中启用或禁用。
