## 文件管理

### 加载图像
您可以通过多种方式加载图像：
- 点击"添加图像"按钮打开文件对话框并选择多个文件。
- 将一个或多个图像文件直接拖放到主窗口上。将出现一个叠加层，允许您选择将它们添加到左侧或右侧列表。
- 按 `Ctrl+V` 从剪贴板粘贴图像。

### 管理图像列表
点击下拉菜单打开图像列表管理器（浮动菜单）。它提供：

- **拖放重新排序：** 点击并拖动项目以重新排序，或将其拖到另一个列表。

- **交换按钮 (⇄)：**
  - 短按仅交换当前选定的两个图像。
  - 长按（约 0.6 秒）完全交换左侧和右侧图像列表。

- **清除按钮 (🗑️)：**
  - 短按仅删除当前选定的图像。
  - 长按清除整个列表。

- **评分系统：** 使用每个图像旁边的 `[+]` 和 `[-]` 按钮来更改其评分。
- **删除项目：** 右键单击任何项目以删除它。


### 列表上的鼠标滚轮
- 在图像列表下拉菜单上滚动鼠标滚轮以切换相应列表中当前选定的项目（左/右）。
- 您也可以在插值下拉菜单上滚动以快速更改重采样方法。

### 评分
- 使用每个图像旁边的 `[+]` 和 `[-]` 按钮来更改其评分。
- 此外，在评分显示上滚动鼠标滚轮以增加或减少当前图像的分数。
