## 快捷键

本页是简短的速查表。详细行为请参见对应主题页面。

### 画布 {#canvas}
- `Ctrl + 鼠标滚轮` — 缩放。
- 缩放高于 `100%` 时使用 `鼠标中键` — 平移。
- `空格键 + 鼠标左键` — 快速预览图像 1。
- `空格键 + 鼠标右键` — 快速预览图像 2。
- `空格键 + Shift + 鼠标左/右键` — 在合并放大镜预览中强制显示某一侧。

### 放大镜 {#magnifier}
- `WASD` — 移动视图或被冻结的放大镜。
- `QE` — 改变放大镜两半之间的间距。

### 文件与保存 {#files-and-saving}
- `Ctrl+V` — 从剪贴板粘贴图像。
- `Ctrl+S` — 使用上次导出设置快速保存。
- `Ctrl+Shift+S` — 打开导出对话框。

### 视频编辑器 {#video-editor}
- `空格键` — 播放 / 暂停。
- `Ctrl+Z` — 撤销。
- `Ctrl+Y` 或 `Ctrl+Shift+Z` — 重做。
- `Delete` / `Backspace` — 删除所选时间轴区间。

### 粘贴叠加层 {#paste-overlay}
- `方向键` 或 `WASD` — 选择粘贴方向。
- `Esc` — 取消粘贴。
