## Canvas Navigation

This page covers only movement inside the canvas and temporary viewing modes.

### Zoom {#zoom}
- Hold `Ctrl` and use the mouse wheel over the canvas.
- Zoom is cursor-centered, so it is easy to focus on a specific detail.

### Pan {#pan}
- When zoom is above `100%`, hold the middle mouse button and drag.
- There is no separate navigation mode; panning happens in the same canvas view.

### Quick Side Preview {#quick-side-preview}
- Hold `Spacebar`.
- `Left Mouse Button` shows Image 1.
- `Right Mouse Button` shows Image 2.
- Release the buttons to return to normal comparison.

### Quick Preview In Combined Magnifier Mode {#combined-magnifier-preview}
- If the active magnifier is combined, hold `Space + Shift`.
- `Left Mouse Button` forces the left side.
- `Right Mouse Button` forces the right side.
- While `Space + Shift` is still held, another click can switch the side again.

### What Changes Temporarily While Zoomed {#zoom-side-effects}
- Filename labels are shown only at **100% zoom**.
- At any other zoom level they are hidden temporarily and return automatically.
- Some overlays may simplify during interactive movement for performance reasons.

### Split Line While Zoomed {#split-line-while-zoomed}
- The split line keeps a visually stable screen position while zoom changes.
- This lets you zoom first and keep comparing without abrupt jumps.

### What Is Configured Elsewhere {#configured-elsewhere}
- Preview quality and interpolation settings are described in **[Settings](help://settings#preview-and-quality)**.
- Non-magnifier comparison behavior is described in **[Comparison](help://comparison)**.
- Magnifier-specific interaction is described in **[Magnifier](help://magnifier)**.
