## Buttons and Controls

Toolbar and in-window panels share one gesture model: a short click, a scroll, and a long-press are different actions on the same control.

### Toolbar habits {#toolbar-habits}

- **Short click** — runs the primary action or toggles the control (show / hide the split, open a panel).
- **Scroll** — over thickness, magnifier size, or list rating nudges the value without a dialog.
- **Long-press** — stronger variant on some icons (clear a whole image list instead of one item).

### Panels vs dialogs {#flyouts}

- **Panel** — in-window chrome (label style, divider options). Click outside or `Esc` to dismiss; nested panels close with their parent.
- **Dialog** — separate window for heavier tasks (export, settings, image properties), same theme as the main window.

### Find a control by name {#find-action}

Press `Ctrl+Shift+P` ({{tr:menu.find_action}}), type part of the label, then run the action or open {{tr:action.palette.learn_more}} / `Ctrl+Enter` when the row has a help topic.
