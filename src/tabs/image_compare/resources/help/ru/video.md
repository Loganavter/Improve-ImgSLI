## Видеоредактор

Запишите живую сессию {{tr:workspace.session_types.image_compare}}, обрежьте таймлайн и закодируйте в видео или GIF. Кадры по-прежнему в [Экспорте](help://export).

### Запись сессии {#recording}

- **Старт / пауза / стоп** — {{tr:image_compare.action.record}} (`R`), {{tr:image_compare.action.pause_recording}}, остановка теми же контролами.
- **Что пишется** — действия на холсте во времени (зум/пан, разделитель, лупа, изображения, настройки) как дорожки контролов с точками записи — не сырой скринкаст.
- **Частота** — [Настройки → Производительность](help://settings#performance) ({{tr:settings.recording_fps}}).

### Открыть редактор {#open-editor}

После остановки откройте {{tr:image_compare.action.video_editor}} (`Ctrl+E`), чтобы листать кадры, править диапазон и экспортировать. Поиск действий окна через {{tr:menu.find_action}} тоже открывает эту тему.

### Правка таймлайна {#timeline}

- **Листать** — маркер времени.
- **Диапазон** — ручки выделения задают отрезок; {{tr:button.trim_to_selection}} оставляет только его.
- **Удалить** — `Delete` / `Backspace`.
- **Воспроизведение** — `Space`; отмена / повтор — `Ctrl+Z` / `Ctrl+Y`.

### Треки и что попало в запись {#tracks}

Дорожки инструментов (разделитель, лупа, вьюпорт и т.п.) показывают, как контролы менялись во времени. Между точками записи редактор плавно склеивает значения.

### Качество превью {#preview-quality}

{{tr:video.preview_quality}} снижает только превью внутри редактора. На финальный файл не влияет.

- **{{tr:video.preview_quality_full}}** — самое чёткое превью; больше нагрузка.
- **{{tr:video.preview_quality_balanced}}** — обычный компромисс для большинства сессий.
- **{{tr:video.preview_quality_performance}}** — легче, если скролл по таймлайну тормозит.
- **{{tr:video.preview_quality_draft}}** — самое быстрое превью; меньше деталей.

### Экспорт видео или GIF {#export-encode}

- **Кадр** — разрешение (при необходимости пропорции), FPS (не выше записи), вписать / обрезать; опциональная заливка пустых краёв.
- **Кодек** — контейнер и кодек на стандартной вкладке; аппаратные — только если система их отдаёт.
- **Качество** — CRF/CQ или битрейт и пресеты.
- **Путь** — файл; кнопки избранной папки подставляют предпочитаемый каталог.
- **Прогресс** — можно остановить; лог показывает сообщения энкодера.
